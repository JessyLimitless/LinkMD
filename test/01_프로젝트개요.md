# 프로젝트 개요

## 배경

AI 코딩 시대에 Claude Code, Cursor 등으로 작업하면 프로젝트당 10~30개의 MD 파일이 생성된다.

## 목적

파편화된 MD 파일을 비즈니스 자산으로 즉시 이식한다.

## 기술 스택

| 구분 | 기술 | 역할 |
|------|------|------|
| Backend | Node.js + Express | API 서버 |
| Engine | Pandoc 3.x | 문서 변환 |
| Frontend | Vanilla JS | 웹 UI |

## 코드 예시

```javascript
const express = require('express');
const app = express();
app.listen(3500);
```
